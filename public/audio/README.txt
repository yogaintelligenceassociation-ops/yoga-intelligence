SOULFUL BACKGROUND MUSIC
========================

To enable the calming site-wide background music, place ONE royalty-free
audio file here named exactly:

    ambient.mp3

Requirements:
  - Format: .mp3  (works on iPhone Safari + all Android/desktop browsers)
  - Mood: soft, soulful flute / bansuri / meditation — looped seamlessly
  - Keep it small-ish (2-5 MB) so it loads fast on phones

Where to get a free, license-safe track (no attribution needed):
  - https://pixabay.com/music/search/bansuri%20meditation/   (download as MP3)
  - https://pixabay.com/music/search/meditation%20flute/

How it works once the file is here:
  - The floating music button (bottom-left) appears automatically.
  - The music starts softly on the visitor's first tap/scroll and loops.
  - Visitors can mute/unmute anytime; their choice is remembered.
  - If this file is absent, the music button simply doesn't show (no errors).

After adding the file: commit + push, and Vercel redeploys it live.
